# Pasta de Imagens

Coloque todas as imagens do site nesta pasta.

Veja o arquivo `IMAGENS.md` na raiz do projeto para a lista completa de imagens necessárias.

## Estrutura Esperada

```
images/
├── hero-home.jpg
├── hero-jr-campo-360.jpg
├── hero-servicos.jpg
├── jr-campo-360.jpg
├── card-1.jpg
├── card-2.jpg
├── card-3.jpg
├── card-4.jpg
├── passo-1.jpg
├── passo-2.jpg
└── passo-3.jpg
```

## Nota

Se as imagens não forem encontradas, o site ainda funcionará, mas as áreas de imagem aparecerão em branco ou com fundo cinza.
