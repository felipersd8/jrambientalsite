Todas as imagens devem ser colocadas na pasta `/public/images/`.

## Formato e Qualidade

- **Formato recomendado**: JPG (para fotos) ou PNG (para gráficos/ícones)
- **Qualidade**: Otimize as imagens para web (70-85% de qualidade JPG)
- **Tamanhos**: As imagens serão redimensionadas automaticamente, mas recomenda-se:
  - Heroes: 1920x1080px (landscape)
  - Cards: 400x300px (4:3)
  - Passos: 300x300px (quadrado)
- **Peso**: Tente manter cada imagem abaixo de 500KB

## Dicas

1. Use imagens de alta qualidade, mas otimizadas para web
2. Todas as imagens devem ser relacionadas ao tema ambiental/agro
3. Considere usar imagens reais de projetos da JR (com permissão)
4. Mantenha consistência no estilo visual (cores, tom, iluminação)
5. Use imagens que transmitam profissionalismo e tecnologia

## Placeholders Temporários

Se você não tiver as imagens ainda, o site funcionará normalmente, mas as imagens não serão exibidas. Você pode:

1. Usar serviços como Unsplash, Pexels para imagens temporárias
2. Criar imagens placeholder usando ferramentas online
3. Usar fotos reais de projetos/equipamentos da JR

## Ferramentas Recomendadas para Otimização

- **TinyPNG/TinyJPG**: Compressão sem perda de qualidade
- **Squoosh**: Ferramenta do Google para otimização
- **ImageOptim**: Para macOS
- **GIMP/Photoshop**: Para edição e redimensionamento
